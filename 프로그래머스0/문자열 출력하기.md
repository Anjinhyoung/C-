```C++
#include <iostream>
#include <string>


int main(void) {
    std::string str;
    std::cin >> str;
    std::cout << str;
    return 0;
}
```

## 여담 

프로그래머스는 보통 ***using namespace std; 와 같이 어떠한 이름 공간**을 사용하는데 나는 적어도 이 레포지토리에서는 프로그래머스 문제를 풀 때 이 코드를 지우고 풀 것이다.
