아직까지는 문제가 너무 쉽다. 백준 파이썬 내공이 있어서 그런지 코드는 다음과 같다.

```C++
#include <iostream>


int main(void) {
    int a;
    int b;
    std::cin >> a >> b;
    std::cout << "a = " << a << std::endl << "b = " << b;
    return 0;
}
```
